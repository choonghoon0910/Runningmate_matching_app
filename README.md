# 🏃‍♀️ RunTogether

**RunTogether**는 실시간 위치 기반으로 러닝 메이트를 찾아주는 **러너 매칭 앱**입니다.  
혼자 운동하는 사람들에게 함께 달릴 수 있는 파트너를 연결해주어,  
건강한 습관과 운동의 즐거움을 동시에 제공합니다.

---

## 📌 프로젝트 개요

> “함께 달리며 건강한 삶과 활력을 되찾는 어플”

많은 사람들이 혼자 운동하기보다는 누군가와 함께하는 것을 선호하지만  
실제로는 **운동 파트너를 찾기 어렵다**는 문제가 있습니다.  
특히 낯선 지역에서 운동하려는 사람들에게는 이 문제가 더욱 크게 다가옵니다.

RunTogether는 이러한 문제를 해결하기 위해,  
**사용자 매칭과 채팅 시스템**을 통해  
가까운 거리의 러너를 자동으로 연결해줍니다.

## ⚙️ 개발 환경

| 구분 | 기술 |
|------|------|
| **Backend** | Spring Boot, H2 Database, IntelliJ IDEA |
| **Frontend (App)** | Android Studio, Java |
| **API** | Kakao Map API |
| **통신 방식** | Retrofit (클라이언트) ↔ Spring Boot (서버) / JSON 기반 데이터 전송 |

---

## 🧠 시스템 구조
[Android Client] ⇄ [Spring Boot Server] ⇄ [H2 Database]
↑
Kakao Map API (실시간 위치 기반)

- Retrofit: 클라이언트에서 JSON 직렬화 및 HTTP 요청 전송  
- Spring Boot: 서버에서 JSON 요청 처리 및 응답 반환  
- H2 Database: 내장형 DB로 별도의 서버 설정 없이 동
